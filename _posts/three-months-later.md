ご無沙汰しています。3ヶ月ぶりです。

その間、生焼け鶏肉カンピロバクター事件で1週間寝込んだり、Claude CodeとCodexのレートリミット2倍キャンペーンに釣られていくつかアプリを作ったりしていました。今メインでやっているプロジェクトを軽く紹介します。

## タスク管理ツール — GitHub IssuesとGistのライト版

GitHub IssuesとGistの要素を組み合わせたツールです。API経由でデータにアクセスできるため、Web・CLI・MCPなど複数のルートから扱えます。

MCPを使えば、ChatGPTやClaudeのチャット画面から「Issue追加しといて」と言うだけで追加できます。

チーム管理や複数プロジェクトへのスケールを意識した設計にしていますが、5回ほど作っては壊した結果、いまはスコープを狭めた自分専用のPreview版を動かしています。UIは映画やゲームの世界観を詰め込んでいます、自分しか使わないし。

## 英語学習ツール — Git/GitHubで使う英語表現を学ぶ

「コミットメッセージを英語で書けるようになりたい」という動機で作ったドリル形式のツールです。単語と例文から問題を自動生成し、四肢選択や穴埋めなど複数パターンに対応しています。

素材には、12万件のOSSコミットを分析して抽出した動詞リストを使いました。

ただし、三日坊主の前科多数なユーザー（私）が最大の阻害要因です。こちらもタスク管理ツールと同じく趣味全開すぎるので、一般公開するなら別ラインを作ろうかなと思ってます。

## MyYomuMoji — 自分に合った読みやすさ設定を共有する

Web UIでフォント・文字サイズ・行間・配色をカスタマイズし、その設定を共有できるツールです。体験を「作る → 渡す → 読む」の3ステップに絞って、実装をシンプルにしました。

センシティブ情報は扱わないため、病気や障害を連想させる表現は避けて「視覚的好み」として扱う設計にしています。設定はAPIから読み込めるので、将来的には他のアプリのユーザー設定として活用する使い方もできるかもしれません。

自分で建てた家の鍵を、自分でなくす。まあ、そういうことです。
